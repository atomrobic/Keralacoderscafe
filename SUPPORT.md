# Support

Need help or not sure where something belongs? Use the guide below.

## Where to Ask

- Bug reports: open a GitHub issue using the bug report template
- Feature ideas: open a GitHub issue using the feature request template
- Security concerns: follow the process in [SECURITY.md](./SECURITY.md)
- Contribution questions: read [CONTRIBUTING.md](./CONTRIBUTING.md)
- Community chat: join Kerala Coders Cafe through
  https://kcc.sh/join/whatsapp

## What to Include

When asking for help, please share:

- What you were trying to do
- What happened instead
- Clear reproduction steps
- Screenshots or console output when relevant
- Your environment details if the problem is platform-specific

## Response Expectations

This is a community-maintained repository, so support is provided on a
best-effort basis. Clear reports and respectful communication help everyone
move faster.
