# Kerala Coders Cafe

A community-first website for Kerala Coders Cafe, built to welcome new developers, highlight contributors, and give the community a strong public home on the web.

## Overview

Kerala Coders Cafe is no longer just a "coming soon" page. The website now presents the community with a full landing experience, a contributor showcase connected to GitHub, a guided WhatsApp onboarding flow, and a cleaner visual identity across desktop and mobile.

## Latest Website Updates

- Replaced the old placeholder page with a full landing page experience.
- Added dedicated sections for the community story, contributors, projects, and joining.
- Built a responsive navigation system with a desktop top bar and mobile bottom nav.
- Added light and dark theme switching with saved user preference.
- Connected the contributors section to the GitHub Contributors API for live public data.
- Added a WhatsApp join gate with quick language-based questions for JavaScript, TypeScript, Python, Java, and C++.
- Improved metadata, Open Graph, JSON-LD schema, and manifest support for better SEO and sharing.
- Configured the app for static export, making deployment easier on static hosting platforms.

## Local Development

1. Clone the repository:

   ```bash
   git clone https://github.com/KERALACODERSCAFE/Keralacoderscafe.git
   cd Keralacoderscafe
   ```

2. Install dependencies:

   ```bash
   npm install
   ```

3. Start the development server:

   ```bash
   npm run dev
   ```

4. Open the app in your browser:

   ```text
   http://localhost:3000
   ```

## Useful Commands

```bash
npm run dev
npm run build
npm run lint
```

`npm run build` generates a static export in the `out/` directory.
